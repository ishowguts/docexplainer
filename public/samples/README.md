# Sample documents for demo

Drop a PDF into this folder and reference it from the presentation demo. Sample
domains to collect:

1. **Lab report** — any pathology test (CBC, lipid profile, HbA1c). You can
   download a free sample from Apollo / Thyrocare / 1mg test reports.
2. **Rent agreement** — a standard 11-month Indian rental agreement template.
3. **Insurance policy wording** — download from any LIC / HDFC Ergo sample pack.
4. **RTI response or govt notice** — any PDF from a government portal.

Keep each under 2 MB for fast demo uploads.
